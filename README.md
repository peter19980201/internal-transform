# 局域网P2P文件传输工具

一个简单易用的局域网点对点文件传输工具，文件直接在设备之间传输，服务器不保存任何文件。

## 特性

- ✨ 点对点直接传输，服务器不存储文件
- 🚀 基于 WebSocket 实现实时通信
- 📱 响应式设计，支持移动设备
- 👥 支持多用户同时在线
- 🔒 仅限局域网内访问，安全可靠
- 📊 实时传输进度显示

## 安装

### 前置要求

- Node.js 14.0 或更高版本

### 步骤

1. 克隆或下载此项目

2. 安装依赖：
```bash
npm install
```

## 使用方法

### 1. 启动服务器

```bash
npm start
```

服务器启动后会显示访问地址：
```
=================================
  局域网P2P文件传输服务已启动
=================================

本机访问: http://localhost:3000
局域网访问: http://192.168.x.x:3000

其他设备通过局域网IP访问此服务
文件直接在设备间传输，不保存在服务器
=================================
```

### 2. 连接服务

- **在同一台电脑上测试**：打开多个浏览器标签页访问 `http://localhost:3000`
- **在局域网内其他设备**：使用显示的局域网IP地址访问，如 `http://192.168.1.100:3000`

### 3. 发送文件

1. 输入昵称，点击"加入房间"
2. 点击上传区域，选择要发送的文件
3. 在左侧用户列表中点击目标用户
4. 等待对方接受文件传输
5. 文件开始传输，可以看到实时进度

### 4. 接收文件

1. 当有人向你发送文件时，会弹出提示框
2. 查看文件信息（文件名、大小、发送者）
3. 点击"接受"开始接收文件
4. 接收完成后文件会自动下载到浏览器默认下载目录

## 技术栈

- **后端**: Node.js + Express + Socket.IO
- **前端**: 原生 HTML + CSS + JavaScript
- **通信**: WebSocket (Socket.IO)

## 工作原理

1. 服务器作为信令服务器，协调用户之间的连接
2. 文件数据通过 Socket.IO 在用户之间直接传输
3. 文件分块传输（64KB/块），支持大文件传输
4. 服务器不存储任何文件数据，只转发信令消息

## 注意事项

- 确保所有设备连接在同一局域网内
- 文件大小限制为 1GB
- 传输速度取决于局域网带宽
- 关闭浏览器标签页后会自动断开连接
- 文件传输过程中请保持网页开启
